# 减肥小助手

一个专为女孩子设计的减肥计划管理工具，集成AI智能分析功能。

## 功能特点

- 📊 体重记录与趋势图表
- 💕 BMI自动计算与健康评估
- 🤖 Claude AI智能分析与建议
- 🎨 温馨的粉色系UI设计

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 使用说明

1. 输入身高和目标体重
2. 每天记录体重数据
3. 点击"获取AI建议"，输入Claude API Key获取个性化分析

## API Key获取

访问 https://console.anthropic.com/ 获取Claude API Key
